    function bareutils.decode_set_cookie ¶

    Summary¶

    Decode a set-cookie header into a dictionary.

    bareutils.decode_set_cookie(set_cookie: bytes)

    Parameters¶

    set_cookie: bytes (optional)

    The set-cookie header

    Returns¶

    Mapping[strAny]: A dictionary of the values

    Description¶

    The max-age value is represented as a datatime.timedelta. The expires value is represented as a datetine.datetime. The secure value is represented as a bool.

    function bareutils.encode_set_cookie ¶

    Summary¶

    Encode set-cookie

    bareutils.encode_set_cookie(name: bytes, value: bytes, *, expires: Union[datetime.datetime, NoneType], max_age: Union[int, datetime.timedelta, NoneType], path: Union[bytes, NoneType], domain: Union[bytes, NoneType], secure: bool, http_only: bool, same_site: Union[bytes, NoneType])

    Parameters¶

    name: bytes (optional)

    The cookie name

    value: bytes (optional)

    The cookie value

    * (optional)

    expires: Union[datetime.datetime, NoneType]

    The time the cookie expires, defaults to None

    max_age: Union[int, datetime.timedelta, NoneType]

    The maximum age of the cookie in seconds, defaults to None

    path: Union[bytes, NoneType]

    The cookie path, defaults to None

    domain: Union[bytes, NoneType]

    The cookie domain, defaults to None

    secure: bool

    Indicates if the cookie is restricted to https, defaults to False

    http_only: bool

    Indicates if the cookie is available to the API, defaults to False

    same_site: Union[bytes, NoneType]

    CORS directive, defaults to None

    Returns¶

    bytes: The set-cookie header
